## 选择服务器

根据网站类型选择合适的服务器厂商，我这里使用的是 `SiteGround`，注册后登录 [SiteGround](https://my.siteground.com/)。

![](/img/wordpress/siteground/10.png)


## 添加站点

首页选择【Websites】并进入👇🏻页面
![](/img/wordpress/siteground/20.png)

点击右上角【NEW WEBSITE】进入👇🏻页面
![](/img/wordpress/siteground/30.png)

由于我自己有域名，所以我选择第二个【Existing Domain】，输入自己的域名，点击【CONTINUE】
![](/img/wordpress/siteground/40.png)


### CREATE EMPTY SITE

点击👇🏻【SKIP & CREATE EMPTY SITE】
![](/img/wordpress/siteground/50.png)

点击👇🏻【FINISH】
![](/img/wordpress/siteground/60.png)
![](/img/wordpress/siteground/70.png)

点击👆🏻【SITE TOOLS】进入👇🏻 站点管理页面
![](/img/wordpress/siteground/80.png)

### Start New Website

如果不通过 【SKIP & CREATE EMPTY SITE】 方式创建站点，也可以通过👇🏻【Start New Website】方式来创建站点

![](/img/wordpress/siteground/90.png)
![](/img/wordpress/siteground/100.png)
![](/img/wordpress/siteground/110.png)
![](/img/wordpress/siteground/120.png)

## 添加域名

点击菜单栏【Domain】->【Subdomains】，在【Create New Subdomain】下的 Name 输入框中输入子域名，然后点击【CREATE】按钮
![](/img/wordpress/siteground/140.png)
![](/img/wordpress/siteground/150.png)

开启子域名管理
![](/img/wordpress/siteground/160.png)

## 安装应用

![](/img/wordpress/siteground/170.png)
![](/img/wordpress/siteground/180.png)

最后点击【INSTALL】按钮安装。安装完成后，还无法立即跳转
![](/img/wordpress/siteground/190.png)
![](/img/wordpress/siteground/200.png)

### 开启HTTPS

打开站点ssl配置页面，选择【IMPORT】
![](/img/wordpress/siteground/250.png)

将证书 xxx.crt 内容复制粘贴到 【Certificate (CRT)】中，将证书 xxx.key 内容复制粘贴到 【Private Key (KEY)】中，最后点击【IMPORT】。
![](/img/wordpress/siteground/260.png)

导入之后启用https，选择站点，点击右边三个小点，点击【Enforce HTTPS】
![](/img/wordpress/siteground/270.png)
![](/img/wordpress/siteground/280.png)

访问站点就会强制携带 https。
