  • 找到 external_url 'http://gitlab.example.com' 这一行，将 http 改为 https

  • vim 搜索 /redirect_http_to_https,将注释去掉，并将值设置为 true, nginx['redirect_http_to_https'] = true

  • 找到这一行 # nginx['ssl_certificate'] = "/etc/gitlab/ssl/#{node['fqdn']}.crt" 修改为 # nginx['ssl_certificate'] = "/etc/gitlab/ssl/gitlab.example.com.crt"

  • 找到这一行 # nginx['ssl_certificate_key'] = "/etc/gitlab/ssl/#{node['fqdn']}.key" 修改为 # nginx['ssl_certificate_key'] = "/etc/gitlab/ssl/gitlab.example.com.key"

  • 找到这一样 # nginx['ssl_dhparam'] = nil # Path to dhparams.pem, eg. /etc/gitlab/ssl/dhparams.pem 修改为 # nginx['ssl_dhparam'] = /etc/gitlab/ssl/dhparams.pem # Path to dhparams.pem, eg. /etc/gitlab/ssl/dhparams.pem

然后保存 :wq 保存退出
