流程控制

PHP遍历数组的三种方式以及各自区别

  1. 使用 for 、while 、do ...while 循环

  2. 使用 foreach 循环

  3. 使用 while、list()、each() 组合循环

    for循环 只能遍历索引数组， foreach 可以遍历 索引数组 和 关联数组， 联合使用 list() , each() 和 while 循环同样可以遍历 索引数组 和 关联数组

    while、list()、each() 组合不会 reset()，reset就是将数组的指针回到开头进行重置，也就是遍历数组的时候不会对数组指针进行操作

    foreach 遍历会对数组进行 reset() 操作，就会对数组指针进行重置

    如果之前数组在对指针进行了相关的操作，然后使用while、list()、each() 对数组进行循环遍历，此时就有可能从数组的中间，从数组现在指针所在位置开始遍历，这样就有可能遍历出来的数组只遍历了一个片段，而 foreach 不会有这样的问题

PHP分支

  1. if…elseif

    在elseif语句中只能有一个表达式为true, 即在elseif语句中只能有一个语句块被执行，多个elseif从句是排斥关系

    使用elseif语句有一个基本原则，总把有限范围小的条件放在前面处理

  2. switch…case…

    和if不同的是，switch后面的控制表达式的数据类型只能是整形、浮点类型或者字符串

    continue语句作用到switch的作用类似与break

    跳出switch外的循环，可以使用continue 2

    switch…case会生成跳转表，直接跳转到对应的case，它不会一层一层去判断，知道值后会直接掉转到所对应的case

    效率： 如果条件比一个简单的比较要复杂得多或者在一个很多次循环中，那么用switch语句可能会快一些

Example

PHP中如何优化多个if…elseif语句的情况？

  1. 把可能性较大的尽量往前挪
  2. 如果判断是一个比较复杂的结构，而且判断的值是整形，浮点型或者字符串可以使用switch…case
