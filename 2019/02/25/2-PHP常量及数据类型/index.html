2-PHP常量及数据类型

Posted on 2019-02-25 Edited on 2020-05-24 In PHP

常量及数据类型的考点

  1. PHP中字符串 可以使用哪三种 定义方法 以及各自的 区别 是什么？

    字符串定义方式：
    单引号 ‘’
    双引号 “”
    heredoc 和 newdoc

    单引号区别：
    单引号不能解析变量
    单引号不能解析转移字符，只能解析单引号和反斜线本身
    变量和变量、变量和字符串、字符串和字符串之间可以使用.连接

    双引号区别：
    双引号可以解析变量，变量可以使用特殊字符和{}包含
    双引号可以解析所有转义字符
    也可以使用.连接

    单引号效率高于双引号

    Heredoc 类型于双引号
    Newdoc 类似于单引号
    两者都用来处理大文本

数据类型

三个数据类型（标量、复合、特殊）

标量： 浮点、整形、字符串、布尔；
复合： 数组、对象
特殊： NONE、RESOURCE

浮点类型：
浮点类型不能运用到比较运算中，不能用于相等的判断里面。如：

布尔类型：
FALSE 的七种情况
0, 0.0, ‘’, ‘0’, false, array(), NULL
整形0, 浮点0.0, 布尔false, 空字符串, 0字符串, 空数组, NULL

数组类型：
超全局数组
$GLOBALS, $_GET, $_POST, $_REQUEST, $_SESSION, $_COOKIE, $_SERVER, $_FILES, $_ENV
其中 $GLOBALS包含了后面的所有内容
$_REQUEST包含了$_GET, $_POST, $_COOKIE
$_REQUEST尽量少用，因为它相当于一个万能钥匙，安全性偏低
$_SERVER:
$_SERVER[‘SERVER_ADDR’]: 服务器端的IP地址
$_SERVER[‘SERVER_NAME’]: 服务器名称
$_SERVER[‘REQUEST_TIME’]: 请求时间
$_SERVER[‘QUERY_STRING’]: 就是url问号后面的一大串，可能为空
$_SERVER[‘HTTP_REFERER’]: 上级请求的页面，从哪里过来的，可能为空比如直接通过网址来访问就是空
$_SERVER[‘HTTP_USER_AGENT’]: 返回头信息里面的agent信息
$_SERVER[‘REMOTE_ADDR’]: 客户端IP地址
$_SERVER[‘REQUEST_URI’]: 如果请求的是index.php 那么这个就是/index.php
$_SERVER[‘PATH_INFO’]: 通常情况下用来处理路由，url中路径部分，假如url: http://www.baidu.com、index.php/use/reg?status=ghost, status=ghost就是query_string,path_info就是/use/reg?status=ghost

NULL
三种情况
直接赋值为NULL, 未定义的变量, unset销毁的变量

常量
定义：
const, define
const更快，是语言结构，define是函数
define不能用于类常量的定义，const可以
常量一经定义，不能被修改，也不能被删除

预定义常量：
FILE, LINE, DIR, FUNCTION, CLASS, TRAIT, METHOD, NAMESPACE

FILE: 返回的文件的路径名和文件名称
LINE： 所在行的行号
DIR： 所在目录
FUNCTION： 所在函数体函数名称
CLASS： 类名
TRAIT： PHP5.4的特性
METHOD： 类名和方法名
NAMESPACE： 返回命名空间名称
