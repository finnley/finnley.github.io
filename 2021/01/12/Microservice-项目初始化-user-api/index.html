初始化项目目录

branch: t1_project_init

目录结构：

  • api: 存在API接口
  • config: 存在项目配置文件
  • forms: 存放项目表单信息
  • global: 存放全局变量
  • initialize: 初始化设置
  • middlewares: 中间件设置
  • proto: proto 文件
  • routers: 路由配置
  • utils: 工具类
  • validator: 自定义验证规则
  • main.go: 主文件
