Docker-LNMP

Posted on 2021-01-02 Edited on 2021-01-03 In Docker

简介

Docker-LNMP 是基于 docker-compose 的运行在 Docker 上的 LNMP 开发环境，包含 PHP, MySQL, Nginx, Redis 等镜像，并支持多版本切换，可以满足一些简单的学习，开发和测试需求

包含镜像

Docker LNMP 包含以下镜像，每种镜像支持多个版本：

  • nginx
  • php-fpm (7.4 - 7.3 - 7.2 - 7.1 - 5.6)
  • mysql (8.0 - 5.7 - 5.6)
  • mongo
  • redis (5.0 - 4.0)
  • memcached (1.5.16 - 1.5 - 1)

其中：

php-fpm 默认是 7.2 版本，如需使用其它版本，配置 .env 文件中 PHP_VERSION 即可；

mysql 默认是 5.7 版本，如需使用其它版本，配置 .env 文件中 MYSQL_VERSION 即可；
