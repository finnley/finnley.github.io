词法分析（Lexical Analysis）

  • main.go
  • print.go
  • go run print.go，会将 main.go 中所有的 token 打印出来

词法分析得到上面的 token 流， token 流最终变成汇编还是不行，还需要先转成某种数据结构才可以用计算机的逻辑去处理，这一步就是语法分析，语法分析就是读入一个一个 token，把它们转成树结构

中间代码（SSA）生成优化

https://golang.design/gossa

SSA(Single Static Assignment)的两⼤ 点是 
• Static: 每个变 只 赋值⼀次(因此应 叫常 更合 ) 
• Single: 每个表 式只 做⼀个简单 算 对于复杂的表 式ab+cd
 拆分成: t0=ab; t1=cd; t2=t0+t1; 三个简单表 式 

上面的 a * 4 + b * c 表达式是无法直接转成汇编的，必须把它们做一次简化成下面每次都是单独计算和赋值语句，然后才能转成汇编

  • 本地使用 GOSSAFUNC=funcname go build xx.go
