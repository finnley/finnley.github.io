2. 准备MySQL配置文件

mysql5.7 安装后的默认配置文件在 /etc/mysql/my.cnf, 而自定义的配置文件一般放在 /etc/mysql/conf.d 这个路径下。
现在我们在本地host主机上自定义的某个目录(如 /data/mysql/conf/ )，先创建两个文件master.conf和slave.conf，分别用于配置主从两个节点。

  • /data/mysql/conf/master.conf
