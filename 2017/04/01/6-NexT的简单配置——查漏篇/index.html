之前写了关于怎么使用GitHub+Hexo搭建个人博客，本来以为结束了，后面才发现只写了一些怎么搭建,配置，上传的过程，下面是一些常用的主题配置。当然这些配置在Next官方文档上都有。

设置菜单

默认主页

菜单配置包括三个部分，第一是菜单项（名称和链接），第二是菜单项的显示文本，第三是菜单项对应的图标。

NexT 使用的是 Font Awesome 提供的图标， Font Awesome 提供了 600+ 的图标，可以满足绝大的多数的场景，同时无须担心在 Retina 屏幕下 图标模糊的问题。

设定菜单内容，对应的字段是 menu。 菜单内容的设置格式是：item name: link 。其中 item name 是一个名称，这个名称并不直接显示在页面上，她将用于匹配图标以及翻译。

比如我想在首页显示 文章分类， 关于我，以及 文章标签，我只需要将各自对应的选项前面的“#”去掉就行了，拿代码来比较，“#”就好像注释掉了这一行，现在想把他显示出来只需要取消注释就行了。

修改后

如果想把站点地图显示出来，只需要把 sitemap 前面的 # 去掉就行了，就这么简单。
