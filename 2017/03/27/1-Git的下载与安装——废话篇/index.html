1-Git的下载与安装——废话篇

Posted on 2017-03-27 Edited on 2020-07-17 In Hexo+Next

前言

以前看到别人有自己的博客很是羡慕，也想有一个属于自己的博客，但是肚子里又没有墨水，生怕没东西写，就算能扯点，也担心不能坚持下去。后来还是忍不住手痒，开始挠了。

我们浏览招聘网站的时候，往往会看到有些公司招聘时会要求附上自己的作品，或者是有自己的Github，或者是自己的技术博客等。作为刚毕业的萌新来说，希望在投递简历的时候，能够给面试增加一些成功的砝码。

下面将介绍我是如何使用 GitHub 配合 Hexo 和 Next 搭建自己博客的。在使用GitHub搭建博客的时候还需要做一些准备，比如需要安装Git, NodeJS等。

下载 Git

进入 Git官网 ，下载适合自己的版本。

点击 Downloads for Windows 进入下载页面。我选择的是64位版本，当然也可以视情况而定。

下载Git工具

windows系统下常用的Git软件有 TortoiseGit 以及 TortoiseGit-LanguagePack语言包。可以进入TortoiseGit 官网进行下载。

我选择的是64位的版本。

下载完后，总共有下面的三个安装包。
